

//El programa inicia solicitando la edad del usuario y tiene un validador:
- Si la edad del usuario es menor a 18, se le informa que el programa es solo para mayores de edad y termina la ejecucion.
- Si el usuario ingresa un valor no numerico, finaliza la ejecucion.
- Si la edad del usuario es igual o mayor a 18 continua la ejecucion y se le solicita un nombre.

//Luego  se proponen 3 actividades: LEER LA MENTE, GENERADOR DE POEMAS, CONVERSOR DE DIVISAS.


//LEER LA MENTE.
Es un juego, se le dan 3 oportunidades al usuario para acertar un valor al azar del 1 al 10
El valor a adivinar es generado al azar en cada ejecucion, y se muestra en la consola del navegador.
El ciclo por respuestas incorrectas solo se repite 3 veces como maximo hasta que el usuario pierde.
Con cada respuesta incorrecta se proporciona una pista a traves de una funcion.

//GENERADOR DE POEMAS
Se solicita al usuario que ingrese una serie de palabras.
Luego esas palabras se incluiran dentro de un relato preconfigurado, elegido al azar de entre
una serie de relatos preconfigurados. La eleccion del relato preconfigurado al azar se hace atando cada
uno de ellos a un numero del 1 al 3, y luego generando un selector con un numero al azar del 1 al 3.

//CONVERSOR DE DIVISAS.
Conversor simple de pesos a dolares o viceversa, con funcion de flecha.
